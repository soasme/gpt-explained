**📚 Read online:** https://www.soasme.com/gpt-explained/

---

## About This Book

This book teaches how GPT-style language models actually work, from the very first step (turning text into integers) to the final step (turning vectors back into words). It is written in the spirit of *Hands-On Machine Learning with Scikit-Learn and TensorFlow* — every concept is explained intuitively first, then backed by math, then grounded in concrete matrix examples, and finally implemented in code.

**The code language is Lisp.** Not because it is the fashionable choice, but because Lisp is the closest thing to executable math. A function is a formula. A list is a matrix row. The code reads like the equations.

**The diagrams are animated.** Every figure in this book is generated by a Manim script, living alongside the prose in `src/manim/`. You can reproduce any animation by running the corresponding script.

## Building Locally

### Prerequisites

```bash
# Rust (for mdBook)
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
cargo install mdbook

# Python (for Manim)
pip install manim
# Also needs LaTeX:
# Ubuntu: sudo apt-get install texlive-full
# macOS:  brew install --cask mactex
```

### Generate images

```bash
cd src/manim
bash render_all.sh --ql      # low quality (fast, ~2 min)
bash render_all.sh --qh      # production quality (~15 min)
```

### Build the book

```bash
mdbook build       # output in book/
mdbook serve       # live-reload dev server at http://localhost:3000
```
